# PCB Etching using Ferric Chloride

This sub-project demonstrates the manual fabrication of a Printed Circuit Board (PCB) using the ferric chloride etching method.

The process involves removing unwanted copper from a copper-clad board, leaving behind only the required circuit tracks.

---

## 🎯 Objective

To understand and implement the chemical etching process for PCB fabrication using simple tools and materials.

---

## ⚙️ Method Used

- Ferric Chloride (FeCl₃) Etching
- Manual track drawing / transfer method

---

## 📁 Contents

- `process.md` → Step-by-step PCB fabrication procedure

---

## 🧪 Key Concept

Ferric chloride reacts with copper and removes excess copper from the PCB surface, preserving only the desired circuit pattern.

---

## ⚠️ Safety Notes

- Use gloves while handling ferric chloride
- Avoid skin contact
- Perform the process in a well-ventilated area
- Dispose of chemicals properly
